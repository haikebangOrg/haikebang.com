# Tabnine 和 Copilot

我最近相当认真地研究和比较了两个主流的 AI 辅助编程服务：Tabnine Pro 和 GitHub Copilot。

基于 Windows PC 的 Emacs，安装过程 Tabnine 完胜，一键安装；GitHub Copilot 借助的 Package Management 是 use package 和 straight.el，与前者相比略有麻烦，不过所需依赖完全可以全部通过 Scoop 安装，所以可以接受。

试用下来 Copilot 就不得了了！

（不确定是不是因为我的 Tabnine 非 Team Pro 套餐）

直接附两张 Copilot 的图，分别为文档和 Emacs Lisp，红框中浅灰字即 Copilot AI 生成的内容和代码。

总结：上 Copilot，不要犹豫，一点都不要。

# Ctrl-c Ctrl-Tab

我的电脑的 CTRL 和 CAPS LOCK 键互换了位置，而 Emacs 有一个快捷键是 Ctrl-c Ctrl-Tab，结果每次按到这组键的时候都有种手指打结了的感觉。
